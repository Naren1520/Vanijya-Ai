# Vanijya AI - Project Overview

## Mission
Empowering India's mandis with AI-powered multilingual communication and fair pricing discovery.

## Key Features

### 1. Multilingual Support
- Real-time translation across 6 Indian languages
- Voice-first design for local vendors
- Cultural context preservation

### 2. AI-Powered Price Discovery
- Smart market analysis
- Fair price recommendations
- Regional price variations

### 3. Negotiation Assistant
- Culturally appropriate phrases
- Counter-offer suggestions
- Local language support

## Technical Architecture

### Frontend
- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS with custom Indian color palette
- **Animations**: Framer Motion for smooth interactions
- **Language**: TypeScript for type safety

### Design System
- **Theme**: Glassmorphic UI with saffron and earth tones
- **Responsive**: Mobile-first approach
- **Accessibility**: WCAG compliant components

### State Management
- React Context for language switching
- Local storage for persistence
- Session-aware loading states

## Pages Structure

1. **Landing Page** (`/`) - Hero section with features overview
2. **Demo Page** (`/demo`) - Interactive AI demo with voice recognition
3. **Dashboard** (`/dashboard`) - Vendor analytics and insights
4. **How It Works** (`/how-it-works`) - Step-by-step process explanation
5. **Impact** (`/impact`) - Success stories and growth metrics

## Development Guidelines

### Code Standards
- Use TypeScript for all components
- Follow Next.js App Router conventions
- Implement proper error boundaries
- Ensure SSR compatibility

### Styling Guidelines
- Use Tailwind utility classes
- Maintain consistent spacing (4px grid)
- Follow Indian-inspired color palette
- Implement glassmorphic effects consistently

### Performance
- Optimize images with Next.js Image component
- Implement proper loading states
- Use dynamic imports for heavy components
- Minimize bundle size

## Deployment
- Platform: Vercel (recommended)
- Environment: Production-ready
- Domain: Custom domain setup
- Analytics: Built-in performance monitoring
